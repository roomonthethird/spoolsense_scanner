Hi all! 

Here is my case for the Spoolsense NFC Holder.

- This uses an ESP-32-S3 Mini and the PN5180 NFC Module.

**For this case, you must solder the boards directly. You cannot use jumpers and headers!**

The reason for this is so the case can be smaller / more space saving. 

The Fusion files are uploaded as well if you want to make any changes.


**Print Settings**

- Standard print settings should work fine, no supports required.

- PLA/PETG, 15-20% infill.

- Case is snap fit, no hardware required. If you need to reopen the case, use a flat edge/spudger to pry it apart.

- Print the lid flat side down.
