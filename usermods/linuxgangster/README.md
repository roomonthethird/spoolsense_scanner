# LinuxGangster's Scanner Mods

## Standalone Case (ESP32-S3-Zero + PN532)

A two-piece case for the ESP32-S3-Zero with PN532 NFC reader.

| File | Description |
|------|-------------|
| `nfc-case-esp32s3zero.3mf` | Main case body |
| `NFC+Reader+Lid+v1.0.stl` | Lid |

### Print Settings

| Setting | Recommended |
|---------|-------------|
| Material | PLA or PETG |
| Layer Height | 0.2mm |
| Infill | 15-20% |

### Hardware

- ESP32-S3-Zero
- PN532 NFC reader

## BoxTurtle PN532 Tray

A modified BoxTurtle tray that holds a PN532 reader alongside an AFC lane. Use this when running multiple scanners for per-lane spool detection.

| File | Description |
|------|-------------|
| `tray_plain_pn532.stl` | Modified BoxTurtle tray for PN532 |

### Hardware

- PN532 NFC reader
- BoxTurtle AFC system
